    test_order() click to toggle source

    Returns the order in which test cases are run.

    ActiveSupport::TestCase.test_order # => :random

    Possible values are :random, :parallel, :alpha, :sorted. Defaults to :random.

    # File vendor/bundle/ruby/2.4.0/gems/activesupport-5.1.5/lib/active_support/test_case.rb, line 38
def test_order
  ActiveSupport.test_order ||= :random
end
    test_order=(new_order) click to toggle source

    Sets the order in which test cases are run.

    ActiveSupport::TestCase.test_order = :random # => :random

    Valid values are:

    • :random (to run tests in random order)

    • :parallel (to run tests in parallel)

    • :sorted (to run tests alphabetically by method name)

    • :alpha (equivalent to :sorted)

    # File vendor/bundle/ruby/2.4.0/gems/activesupport-5.1.5/lib/active_support/test_case.rb, line 28
def test_order=(new_order)
  ActiveSupport.test_order = new_order
end
