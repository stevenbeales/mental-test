    def_private_delegators(target, *private_delegates) click to toggle source

    Since Ruby 2.4, Forwardable will print a warning when calling a method that is private on a delegate, and in the future it could be an error: bugs.ruby-lang.org/issues/12782#note-3

    That's why we revert to a custom implementation for delegating one private method to another.

    # File vendor/bundle/ruby/2.4.0/gems/pry-0.11.3/lib/pry/forwardable.rb, line 14
def def_private_delegators(target, *private_delegates)
  private_delegates.each do |private_delegate|
    define_method(private_delegate) do |*a, &b|
      instance_variable_get(target).__send__(private_delegate, *a, &b)
    end
  end
  class_eval { private(*private_delegates) }
end
