    class Parser::Context

    Context of parsing that is represented by a stack of scopes.

    Supported states: + :class - in the class body (class A; end) + :sclass - in the singleton class body (class << obj; end) + :def - in the method body (def m; end) + :defs - in the singleton method body (def self.m; end) + :block - in the block body (tap {}) + :lambda - in the lambda body (-> {})
