    type() click to toggle source

    Type of this comment.

    * Inline comments correspond to `:inline`:

      # whatever

* Block comments correspond to `:document`:

      =begin
      hi i am a document
      =end

    @return [Symbol]

    # File vendor/bundle/ruby/2.4.0/gems/parser-2.5.0.0/lib/parser/source/comment.rb, line 75
def type
  if text.start_with?("#".freeze)
    :inline
  elsif text.start_with?("=begin".freeze)
    :document
  end
end
