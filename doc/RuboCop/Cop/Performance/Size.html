    class RuboCop::Cop::Performance::Size

    This cop is used to identify usages of `count` on an `Array` and `Hash` and change them to `size`.

    @example

    # bad
[1, 2, 3].count

# bad
{a: 1, b: 2, c: 3}.count

# good
[1, 2, 3].size

# good
{a: 1, b: 2, c: 3}.size

# good
[1, 2, 3].count { |e| e > 2 }

def autocorrect(node)

end
