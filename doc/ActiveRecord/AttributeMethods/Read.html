    read_attribute(attr_name, &block) click to toggle source

    Returns the value of the attribute identified by attr_name after it has been typecast (for example, “2004-12-12” in a date column is cast to a date object, like Object.new(2004, 12, 12)).

    # File vendor/bundle/ruby/2.4.0/gems/activerecord-5.1.5/lib/active_record/attribute_methods/read.rb, line 50
def read_attribute(attr_name, &block)
  name = if self.class.attribute_alias?(attr_name)
    self.class.attribute_alias(attr_name).to_s
  else
    attr_name.to_s
  end

  name = self.class.primary_key if name == "id".freeze && self.class.primary_key
  _read_attribute(name, &block)
end
