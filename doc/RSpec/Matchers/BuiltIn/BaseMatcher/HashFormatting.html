    improve_hash_formatting(inspect_string) click to toggle source

    `{ :a => 5, :b => 2 }.inspect` produces:

    {:a=>5, :b=>2}

    â€¦but it looks much better as:

    {:a => 5, :b => 2}

    This is idempotent and safe to run on a string multiple times.

    # File vendor/bundle/ruby/2.4.0/gems/rspec-expectations-3.7.0/lib/rspec/matchers/built_in/base_matcher.rb, line 151
def improve_hash_formatting(inspect_string)
  inspect_string.gsub(/(\S)=>(\S)/, '\1 => \2')
end
