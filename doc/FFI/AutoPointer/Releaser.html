    class FFI::AutoPointer::Releaser

    @abstract Base class for {AutoPointer}'s releasers.

    All subclasses of Releaser should define a +#release(ptr)+ method.

    A releaser is an object in charge of release an {AutoPointer}.

def free
  if @ptr
    release(@ptr)
    @autorelease = false
    @ptr = nil
    @proc = nil
  end
end
